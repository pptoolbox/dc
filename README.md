# DC - Desktop Calendar

Time is invaluable. But sometimes it passes before we even know. This will just show you how much days has passed in the current year so you will be able to put more effort into your goals. Hope this will add a little bit of value in your life. Best of lucküëç

## Features

- **Full Year Calendar**: All 12 months displayed in a clean grid layout
- **4K Resolution**: Generated at 3840√ó2160 for sharp, modern displays
- **Smart Date Coloring**: Highlighted current day along with grayed out past days
- **Monospace Typography**: Clean, modern DM Mono font
- **Automatic Date Detection**: Always highlights the current date

## Requirements (Installed automatically)

- Python 3.x
- PIL/Pillow (Python Imaging Library)
- DM Mono font (included/available at `/opt/dc/DMMono-Medium.ttf`)

## Installation

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/pptoolbox/dc/main/install.sh)"
```

The installation script sets up the necessary files and dependencies, and places the wallpaper in `/usr/local/share/wallpapers`.

## Usage

After installation, you should find the wallpaper in your settings under wallpapers.

If not detected automatically, you can also set your wallpaper from `/usr/local/share/wallpapers`
